# MPC-TLS SDK


## 概述

欢迎使用`Primus MPC-TLS SDK` ！您可以使用[SDK](https://github.com/primus-labs/mpctls-js-sdk?tab=readme-ov-file)在`dApp`中直接实现**数据验证解决方案**的工具，而无需后端加密或证明工作流程开发。
